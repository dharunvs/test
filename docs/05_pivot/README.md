## Purpose

This folder defines the v0.1 hard pivot to a single wedge: **AI Intent Timeline**.

During v0.1, these docs override broad-scope platform plans.

## Locked Decisions

1. Wedge only: onboarding + intent capture + task-scoped timeline.
2. Non-wedge runtime scope uses **code removal** (not hide-only nav gating).
3. Timeline scope is **task-scoped only** in v0.1 (no project aggregate feed).
4. LLM/automation updates are out of scope unless they improve this wedge loop.
